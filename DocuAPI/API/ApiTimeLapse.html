Dar de alta Timelapse
http://trianasat2-salesianostriana.rhcloud.com/timelapse
 Esta petción permite dar de alta un Timelapse. Si ya existe devuelve código 409. Si todo es correcto devuelve un código 201. 		POST
  • fecha_captura: es la fecha en la que se tomó la foto.
  • fecha_subida: es la fecha en la que se sube la foto.
  • foto: la url de la foto.
  • estado_bateria: es el estado de la bateria.
