PROYECTOS http://trianasat2-salesianostriana.rhcloud.com/proyectos
Esta función sirve para crear un nuevo proyecto 		POST
  • descripcion: Descripción del proyecto
  • localidad: Localidad donde se desarrolla el proyecto
  • fecha_creacion: Fecha de creación del proyecto
  • fecha_lanzamiento: Fecha de lanzamiento del proyecto
  • nombre: Nombre del proyecto
  • token: token del proyecto
  • api_key: Credenciales que necesita la aplicacción
  • api_secret: Credenciales que necesita la aplicacción
  • access_token: Credenciales que necesita el usuario
  • access_token_secret: Credenciales que necesita el usuario
