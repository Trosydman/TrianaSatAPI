Dar de alta Usuario
http://trianasat2-salesianostriana.rhcloud.com/usuarios
 Esta petción permite dar de alta un usuario Si ya existe devuelve código 409. Si todo ok devuelve un código 201. 		POST
  • nombre: es el nombre del usuario.
  • apellidos: es el apellido del usuario.
  • email: es el mail del usuario.
  • password: es la contraseña del usuario.
  • administrador: es un booleano que indica si es o no admin.
  • organización: la url + id de la organización.
