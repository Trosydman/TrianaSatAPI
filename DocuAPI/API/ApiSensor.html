Dar de alta Usuario
http://trianasat2-salesianostriana.rhcloud.com/datossensores
 Esta petción permite dar de alta un usuario Si ya existe devuelve código 409. Si todo ok devuelve un código 201. 		POST
  • fecha: es la fecha de recogida del dato.
  • altirud: es el altitud del dato.
  • temperatura: es la temperatura.
  • presión: es la presión.
  • calidad_aire: es la calidad del aire
  • humedad: es la humedad
